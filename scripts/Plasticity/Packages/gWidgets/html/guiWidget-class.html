Class "guiWidget" ~~~

Description

Base class for gWidget objects

Objects from the Class

Objects can be created by calls of the form new("guiWidget", ...).

These objects have two slots: a widget provided by a guiToolkit and a toolkit. Method dispatch is done on both values.

Slots

toolkit:

Object of class "guiWidgetsToolkit". A specification of which GUI toolkit the widget uses.

widget:

Object of class "ANY". A widget returned by the corresponding toolkit function.
